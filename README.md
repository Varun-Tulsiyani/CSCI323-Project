# CSCI323 Project

## Jupyter Notebooks

* **BASELINE_MODEL_RANDOM_FOREST_REGRESSION.ipynb** – Builds and evaluates a baseline Random Forest regression model for wave energy prediction.
* **Baseline_Models_Ridge_Lasso_Wave_Energy.ipynb** – Implements Ridge and Lasso regression models as baseline linear approaches.
* **advanced models with data processor.ipynb** – Experiments with advanced machine learning models integrated with a data processing pipeline.
* **data_engineering.ipynb** – Performs data cleaning, transformation, and feature engineering for the dataset.
* **optimization_run_notebook.ipynb** – Runs optimization experiments using the genetic algorithm–based optimization module.
* **visualization.ipynb** – Generates plots and visualizations for exploratory data analysis and model results.

## Python Scripts

* **main.py** – Entry point script used to run the training, evaluation, or optimization workflows.
* **linear_models.py** – Implements linear regression–based models such as Ridge and Lasso.
* **tree_models.py** – Contains tree-based models such as Random Forest and related utilities.
* **nn_models.py** – Defines neural network architectures and training procedures.
* **pipelines.py** – Builds machine learning pipelines that combine preprocessing and model training steps.
* **transformers.py** – Custom data transformers used for preprocessing and feature transformations.
* **optimization_ga.py** – Implements a genetic algorithm for optimization of model parameters or system configurations.
* **utils.py** – Utility helper functions used across different modules in the project.
